# SMART-MICROGRID-RENEWABLE-ENERGY-MANAGEMENT-SYSTEM-

The Smart IoT Microgrid & Renewable Energy Management System is an embedded systems project designed to simulate a modern, intelligent power distribution network using an ESP32 microcontroller in the Wokwi environment. It integrates renewable energy generation, energy storage, and load management into a single automated system. The project uses sensors, actuators, and IoT communication to monitor and control power flow in real time.

The system simulates key smart grid functions such as solar tracking, load monitoring, battery charge control, and automatic switching between solar, battery, and utility grid sources. It also provides real-time data visualization through an LCD display and remote monitoring via cloud connectivity.
